WMS Website

The website is designed for a pick-to-light (PTL) system. When warehouse employees receive an order, it is displayed on the website and indicates the warehouse location for picking the order. I developed configuration website for pick-to-light (PTL) system which is a part of Warehouse Management System (WMS). I utilized React, CSS, and HTML to create intuitive and functional user interface (UI) for an industrial environment according to design requirements.
